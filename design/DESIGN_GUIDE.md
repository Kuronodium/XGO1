このドキュメントはXGOの新UIデザイン（ターン強調レイアウト）の指針をまとめたものです。

# XGO UI Design Guide

## 1. 参照モック

- `design/turn_ui_mock.html` を基準にする
- 実装前の検証はこのモックで行う

## 2. 目的・印象

- ダークトーンを基調にし、盤面と石のコントラストを強くする
- どちらの手番かを **一目で分かる** UI にする
- 余計な情報を減らし、盤面への集中を促す

## 3. レイアウト構成

### 3-1. トップバー（情報）

- 左右に捕獲数と小石のチップ列を配置
- 中央に `TURN xx` を配置（文字間隔を広めに）
- 手番側のブロックは背景を明るくする（アクティブ強調）

### 3-2. 中央エリア

- 盤面は中央に固定し、左右に「手番サイド」を置く
- 手番サイドは背景を明るくする（ホワイトグロー系）
- サイドには大きな石（黒 or 白）と TURN ラベルを表示

### 3-3. ボトムアクション

- `Show Log` / `Finish Game` / `Reset` を横並びに配置
- 角丸で凹凸感のあるボタンにする

## 4. 盤面の表現

- 9x9 の格子を **交点中心で描画** する
- 盤面はダークな矩形＋淡いグリッド線で構成
- 盤面外枠はソフトなシャドウで浮かせる

## 5. 石の表現

- 白石：明るいグラデーション＋柔らかい影
- 黒石：深い黒＋白い外形線＋強めの影（視認性優先）
- ホバー時はわずかに拡大（0〜4%程度）

## 6. 障害物（X）の表現

- 交点上に「×」の線を描画
- 明るい線で盤面から浮かせる
- 石と混同しないサイズ・明度にする

## 7. 状態による色の切り替え

- **手番側の背景が明るくなる**
- トップバーの捕獲ブロックも手番側を明るくする
- 非アクティブ側は控えめな明度で統一する

## 8. モーダル

- Setup / Result / Event Log はモーダルで表示
- 背景は暗くブラー/半透明で抑え、主役はモーダルに

## 9. 実装へのメモ

- レイアウトは `design/turn_ui_mock.html` の比率を優先
- 手番強調の色は `--color-*` トークンで制御
- 盤面は `background-position` と `spacing` を用い交点整列を維持
